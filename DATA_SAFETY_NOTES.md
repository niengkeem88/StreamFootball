# Data Safety Notes

- Local preferences: stored on device with DataStore.
- Favorites: stored on device as match IDs.
- Ads: Google Mobile Ads SDK included and disabled unless configured.
- Backend: football data fetched from configured API URL.
- Streaming: no unauthorized streams, no IPTV/M3U scraping, no user-submitted stream links.
- Accounts: no login or account system in v1.
